# OSRS Burialbound

A local-first companion app for the **Burialbound** Old School RuneScape restrictive playthrough concept.

Burialbound makes every major unlock pass through a memorial rite. Before using a new item, boss, region, activity, quest reward, or tier, the account must bury what it is replacing or surpassing.

## First Prototype

- Create pending unlocks.
- Generate burial rites from unlock type.
- Track offerings and rite clauses.
- Approve legal unlocks after a rite is completed.
- Keep a memorial archive and breach notes.
- Export and import local save files.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

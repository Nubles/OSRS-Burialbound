# OSRS Burialbound

A local-first companion app for the **Burialbound** Old School RuneScape restrictive playthrough concept.

Burialbound makes every major unlock pass through a memorial rite. Before using a new item, boss, region, activity, quest reward, or tier, the account must bury what it is replacing or surpassing.

## Companion Features

- Start with an onboarding walkthrough that explains the Burialbound loop.
- Use a tabbed command center for drafting, active rites, registers, archive, and codex.
- Follow a next-up strip for the current rite, open atonement, and ready burials.
- Set account style, run phase, and strict ledger mode.
- Generate rites by unlock type and power tier.
- Track what each unlock surpasses, who witnessed it, and what proves first legal use.
- Keep pending unlocks explicitly forbidden until their rite is buried.
- Complete generated offerings and clauses before approval.
- See reveal feedback when rites are generated, breaches are logged, and unlocks are buried.
- Review legal unlocks, forbidden unlocks, run stats, and recent chapel history.
- Record breaches with severity, penalty, and atonement status.
- Search the memorial archive.
- Export and import local save files with migration support and local backup snapshots.
